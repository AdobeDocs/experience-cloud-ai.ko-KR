# Data Engineering Agent

&lt;!— 초안입니다 —>

Data Engineering Agent은 Adobe Experience Platform의 데이터 팀을 위한 Agent Orchestrator 기반 AI 기반 보조 파일럿입니다. 이 에이전트는 데이터 모델링, 온보딩, SQL 데이터 준비, 거버넌스 및 라이프사이클 관리와 같은 전체 데이터 라이프사이클에서 과도한 작업을 처리하도록 설계되었습니다. Data Engineering Agent을 사용하면 엔지니어와 설계자가 데이터 품질이 향상되고 수동 작업이 줄어들어 더 빠르게 이동할 수 있습니다.

Data Engineering Agent은 워크플로우를 최적화하는 데 활용할 수 있는 기술로 구성되어 있습니다.

| Data Engineering Agent 스킬 | 설명 |
| --- | --- |
| 데이터 온보딩 | <strong>데이터 온보딩</strong>은(는) S3, Marketo 및 DLZ(데이터 랜딩 영역)와 같은 배치 소스를 지원합니다. 다음과 같은 기능을 제공합니다. <ul><li>S3, 데이터 랜딩 영역, Marketo 등을 포함하여 다양한 데이터 소스에 연결.</li><li>소스 데이터(예: 고유 값, null, 중복 및 기본 품질 지표)를 프로파일링하는 중입니다.</li><li>소스 필드를 표준 XDM 필드 그룹과 사용자 정의 필드 모두에 정렬합니다.</li><li>데이터 수집 데이터 흐름 제안 및 구축(예: S3 또는 Marketo에서 RTCDP/CJA 데이터 세트로 데이터 이동).</li></ul> |
| 자동화된 데이터 품질 및 의미 있는 데이터 보강 | <ul><li>예외 항목을 식별하고 유효/잘못된 레코드 수를 보고하는 등 포괄적인 데이터 품질 평가를 수행합니다.</li><li>유형 표준화, 통화 형식 지정 및 ID 중복 제거와 같은 데이터 품질 개선을 권장합니다.</li><li>데이터 샘플링 및 지능형 이름 지정 패턴 인식을 기반으로 스키마에 의미 체계 강화를 적용합니다.</li><li>제안된 보강에 대한 확인을 위해 사용자를 참여시켜 최종 승인된 데이터 변경 사항을 확인합니다.</li></ul> |
| Data Distiller | |
| 데이터 수집 | |
| 데이터 유효성 검사 | |

Data Engineering Agent을 사용하여 다음을 확인할 수 있습니다.

- **더 빠른 온보딩**: 몇 주 동안 수동 스키마 매핑 및 파이프라인을 설정하던 시간을 몇 시간 동안 가이드 및 대화형 구성으로 단축합니다.
- **높은 데이터 품질**: 기본 제공 데이터 품질 분석 및 의미 체계 검사로 인해 잘못된 레코드 및 프로덕션 인시던트가 줄어듭니다.
- **향상된 거버넌스 및 규정 준수**: 거버넌스 정책은 나중에 생각하는 것보다 디자인 타임(레이블, TTL, ID 처리)에 첨부됩니다.
- **더 생산적인 데이터 팀**: 반복적인 SQL/데이터 흐름 작업이 자동화되므로 엔지니어는 더 높은 가치의 설계 및 최적화에 집중합니다.
- **광범위한 셀프 서비스**: 전문가가 아닌 이해 당사자는 보호 기능이 있는 일반적인 데이터 준비 작업을 안전하게 셀프 서비스할 수 있습니다.
